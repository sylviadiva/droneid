This WooCommerce extension supports <strong>slider and carousel layout</strong> and features <strong>lots of customization options.</strong> It is a ready-to-go plugin packed with <strong> 5+ beautiful pre-designed templates.</strong> </p>

The extension offers product filter feature (can be filtered on the basis of taxonomies and category) and product sorting option - which makes your eCommerce site more convincing. It is simple and easy-to-use plugin that allows you to add unlimited product carousel/slider anywhere in your website via shortcode. This WooCommerce extension works flawlessly with all the WordPress themes and is well tested across wide range of devices available.

== Installation ==

1. Unzip product-slider-for-woocommerce-lite.zip
1. Upload all the files to the /wp-content/plugins/product-slider-for-woocommerce-lite
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. For customizing the plugin’s settings, click on Product Slider For WooCommerce menu in WordPress left admin menu.
To display the Product Slider For WooCommerce in the frontend, please use automatically generated shortcode from Product Slider For WooCommerce Usage metabox.


== Frequently Asked Questions ==

= Is this plugin compactible with all themes? =
Yes, this plugin is compactible with all themes which have followed the wordpress theme development standards.

= Do I have to have HTML and CSS Knowledge? =
No you don't need to have knowledge of HTML and CSS unless if you want custom works other than the pre designed templates.

= How do I use the Product Slider For WooCommerce? =
You just need to create a Product Slider from backend and then a shortcode is available in usages metabox. Just copy and paste the shortcode in any page,post or templates.

= How many product sliders can be created with a plugin? =
You can create product's slider as many as you want.

= Can I use multiple product sliders in a single post or page? =
Yes, You can use multiple product sliders in a single page/post.
